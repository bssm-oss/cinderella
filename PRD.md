# PRD — Cinderella

## 목적
Cinderella는 근무 시간이 지났음에도 계속 작업하는 사용자에게 유머러스하고 점진적으로 귀가를 권장하여 워라밸을 회복하도록 돕는 macOS 앱입니다. 연구/데모/학습 목적을 우선으로 하며, 실제 배포 전 법적·윤리적 검토가 필요합니다.

## 문제 정의
- 사용자가 과도하게 잔업을 하며 건강과 생산성이 저하됨
- 기존 알림은 무시되기 쉬움

## 목표
- 사용자가 정해진 시간에 퇴근하도록 친근한 방식으로 유도
- 사용자 동의와 투명성 보장(설정에서 모든 효과 개별 토글 가능)
- 안전장치(핫키) 제공

## 핵심 기능(제품 요구사항)
1. 상태바(툴바)에서 퇴근시간 표시 및 Start/Stop 제어
2. 사용자가 설정한 퇴근시간 이후 동작 시작
3. 1분 단위 타이머, 10분 단위로 intensity 증가
4. 30분/60분 단위로 새로운 이벤트 추가
5. 개별 이벤트 토글, 전체 비활성화 옵션
6. 즉시 종료 핫키(예: Cmd+Opt+Ctrl+Shift+ESC)
7. 접근성 권한 설명 및 동의 플로우

## 사용자(페르소나)
- 과중 노동을 경고받기 원하는 직장인
- 데모/연구자(앱 동작을 테스트하려는 개발자)

## 성공 지표
- 사용자 설정 퇴근시간에 대한 알림 응답률(클릭/종료)
- 강도 단계당 사용자의 중단 비율(예: 10분 내 중단률)

## 비기능 요구사항
- macOS AppKit 기반으로 경량 동작
- 접근성 권한 사용 시 보안·프라이버시 준수
- 사용자 데이터(카메라 등)는 옵트인 후 로컬 처리, 저장 금지

## 출시 전 마일스톤
- M1: 상태바 UI + 설정 저장 (UserDefaults)
- M2: EventScheduler(타이머/강도) + 기본 이벤트(사운드, 커서 지터)
- M3: 추가 이벤트(입력 교란, 창 최소화) + 안전장치
- M4: 테스트 및 문서화
