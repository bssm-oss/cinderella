# Cinderella

## Install (for users)
1. Go to the Releases page:
   - https://github.com/bssm-oss/cinderella/releases
2. Download `Cinderella.dmg`.
3. Open the DMG.
4. Drag `Cinderella.app` into `Applications`.
5. Run `/Applications/Cinderella.app`.

## Required macOS Permissions
Open `System Settings` -> `Privacy & Security` and allow `Cinderella` in:
- `Accessibility`
- `Input Monitoring`

After granting permissions, quit and relaunch Cinderella.
